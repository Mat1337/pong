# pong-clone
clone of the pong game written in pure c

# building

```
    git clone https://github.com/Mat1337/pong-clone.git --recurse-submodules
```

```
    cd pong-clone
    mkdir build
    cd build
    cmake ..
    cmake --build . --config Release
```
